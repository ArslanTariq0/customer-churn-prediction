# Customer Churn Prediction (AI Term Project – Idea #36)

Predicts if a telecom customer will churn (leave) using Random Forest ML.

## Features
- Streamlit web demo
- Trained on Telco Customer Churn dataset (Kaggle)
- Shows churn risk % + retention suggestions

## How to Run
1. Clone: `git clone https://github.com/YOUR_USERNAME/customer-churn-prediction.git`
2. `cd customer-churn-prediction`
3. `python -m venv venv && source venv/bin/activate`
4. `pip install -r requirements.txt`
5. `streamlit run app.py`
